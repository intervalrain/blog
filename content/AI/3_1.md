TensorFlow 是一個開源的 Python 機器學習框架，主要由 Google 開發。  
TensorFlow 可以做到以下的事：
+ 可進行自動微分、計算梯度。
+ 可在 CPU 上運行，也可以在 GPU 及 TPU 上運行。
+ 可以將運算程序分散到多台機器上共同執行。
+ 可以匯出為各種不同語言的程式，包含 C++, JavaScript 或 TensorFlow Lite，使得 TensorFlow 應用可以輕鬆部署到各種實際場景。
