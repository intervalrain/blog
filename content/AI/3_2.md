Keras 是建立在 TensorFlow 上的 Python 深度學習 API，提供了簡易方法來定義、訓練深度學習的模型。  
可以想成 TensorFlow 負責張量運算、Keras 是演算法。

1. Keras（高層 API）：
    + 負責深度學習模型的高階抽象
    + 提供用戶友好的介面
    + 處理模型定義和訓練流程
    + eg. Layer, Model, Optimizers, Loss functions, Metrics

2. TensorFlow（中層）：
    + 處理底層的數學運算
    + 管理計算圖和自動微分
    + 優化運算效率
    + eg. Tensor, Variable、GradientTape

3. Hardware（硬體層）：
    + 實際執行計算任務
    + 提供不同的計算加速選項
    + 優化特定類型的運算
    + eg. CPU, GPU, TPU
