1. 安裝 Anaconda
2. 創建虛擬環境
```
conda create -n tensorflow
```
3. 進入虛擬環境 (macOS/Linux)
```
source activate tensorflow
```
4. 在環境內安裝 tensorflow
```
pip install tensorflow
```
5. 在環境內安裝 jupyter notebook
```
pip install jupyter notebook
```
6. 在環境內安裝 pandas
```
pip install pandas
```
7. 開啟 jupyter notebook
```
jupyter notebook
```
