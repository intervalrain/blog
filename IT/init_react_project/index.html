[IT] 初始 react project (TailwindCSS & TypeScript & Context)

November 28, 2024 · 1 分鐘 · Rain Hu | Suggest Changes
  1. 建立專案
npx create-react-app [project-name] --template typescript
cd [project-name]
  1. 安裝 TailwindCSS 及其相依套件：
bashCopynpm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
  1. 設定 tailwind.config.js：
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{js,jsx,ts,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
  1. 在 src/index.css 中加入 Tailwind directives：
@tailwind base;
@tailwind components;
@tailwind utilities;
